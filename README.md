# My Recipes Website

Hello everyone, and welcome to my first ever project, a recipes website!

This project is an assignment from the TOP's (The Odin Project) curriculum to culminate with the beginner html section. To learn more about the TOP's curriculum, [click here](https://www.theodinproject.com/about).

With this project, I hope to display all the html tools that I have learned from the curriculum, such as html elements, attributes, and values, and to also reflect on all the information that I was able to retain and employ in this project, as well as to learn in the process.

Needless to say, if you decided to look into my project, I hope you have a pleasant (or humorous) time looking through it!
